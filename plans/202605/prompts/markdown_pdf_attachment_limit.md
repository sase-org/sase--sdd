  When an agent modifies more than 10 markdown files, can we stop Telegram and Google chat from converting them to PDFs and sending them to the user? We should only do this when the number of markdown files changed is 10 or less. Otherwise, we should send a message to the user letting them know how many markdown files were edited. Think this through thoroughly and create a plan using your `/sase_plan` skill before making any file changes.
