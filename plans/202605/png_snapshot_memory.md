## Context

## Research Findings

- `Justfile` routes `just test` and `just test-cov` through `_setup-visual` and `tools/run_pytest` with `-m "not slow"`,
  which includes tests marked `visual`; `just test-visual` runs only `-m visual`.
- The ACE visual suite is under `tests/ace/tui/visual/`, with committed PNG goldens in
  `tests/ace/tui/visual/snapshots/png/`.
- `tests/ace/tui/visual/png_diff.py` renders Textual SVG exports to PNG with CairoSVG, compares pixels with Pillow,
  writes actual/expected/diff/source artifacts under `.pytest_cache/sase-visual/`, and supports
  `--sase-update-visual-snapshots` to intentionally refresh goldens.
- Local runs default to exact pixel equality; GitHub Actions workflow runs allow a small ratio-only tolerance for
  renderer drift. Individual snapshots can also set explicit tolerances.
- `tests/ace/tui/visual/conftest.py` pins color and fontconfig/Fira Code for deterministic rendering.
- CI has a separate `visual-test` job that runs `just test-visual` and publishes a failure report generated by
  `tools/render_visual_snapshot_failure_report`.
