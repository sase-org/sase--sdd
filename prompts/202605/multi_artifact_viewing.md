 Can you help me add support to the artifact panel for selecting multiple documents to view?

- The user should be able to use a new `m` keymap to mark the artifacts he wants to open.
- After pressing `<enter>`, the first document will be shown in the terminal.
- If multiple artifacts were selected and the document we are viewing is not the last, we should make an `N` (next
  artifact) option available.
- If multiple artifacts were selected and the document we are viewing is not the first, we should make a `P` (prev
  artifact) option available.

Think this through thoroughly and create a plan using your `/sase_plan` skill before making any file changes.
